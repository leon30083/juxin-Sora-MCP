# 《项目部署手册》

## 环境准备
- 系统：Windows 11。
- 环境变量：`API_BASE_URL`、`API_KEY`、可选 `API_KEY_HEADER`。

## 配置
- 在本地运行前设置环境变量，`API_KEY_HEADER` 默认使用 `Authorization`（Bearer）。

## 使用说明
- 通过 MCP 客户端加载服务器并调用工具：
  - `create_character`：传入 `video_url` 或 `video_base64`。
  - `generate_video`：传入 `prompt`、`model`，可选 `character_id`。
  - `get_task_status`：传入 `task_id`。

## 验证
- 参考 `tests/manual-cases.md` 执行端到端验证。
