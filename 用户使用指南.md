## 🚀 开始创作

### 基础使用：文生视频

只需要用自然语言描述你想要的视频：

```
请帮我创建一个视频：
内容：一只可爱的柴犬在公园草地上奔跑，阳光明媚
格式：横屏（16:9）
时长：15秒
```

### 进阶使用：图生视频

如果你有参考图片，可以让 AI 基于图片生成视频：

```
请用这张图片创建视频：[图片URL]
描述：让照片中的人物微笑并向镜头挥手
其他要求：横屏，1080p，不要水印
```

### 实用技巧

1. **描述要具体**：
   ```
   不好：做一个猫的视频
   好：创建一个视频：一只橘猫坐在沙发上，懒洋洋地伸懒腰，阳光从窗户照进来
   ```

2. **指定画面风格**：
   ```
   创建一个宫崎骏风格的视频：小精灵在森林里飞舞，萤火虫点点，15秒横屏
   ```

3. **描述动作细节**：
   ```
   生成一个视频：咖啡杯被拿起，热气袅袅升起，背景是冬日的雪景，无水印
   ```

4. **关于默认设置**：
   - 默认横屏（16:9），适合宽屏显示
   - 默认无水印，视频更纯净
   - 默认15秒时长，内容更丰富

## 📊 了解视频生成过程

视频生成需要一点时间，就像魔法需要酝酿一样：

1. **提交任务**：告诉 AI 你想要什么
2. **排队等待**：系统会安排你的任务
3. **开始生成**：AI 开始创作
4. **完成生成**：视频准备好了！

### 查询进度

你可以随时询问：
```

```

或者使用任务ID查询：
